# 拉伸法测金属丝杨氏模量网页工具

这是一个本地静态网页项目，用于处理“拉伸法测金属丝的杨氏模量”实验数据。

在线访问地址：

http://turtleneck.top/

## 文件说明

- `index.html`：网页结构，直接双击打开即可使用。
- `style.css`：网页和打印样式。
- `app.js`：数据处理、最小二乘拟合、不确定度计算和 Canvas 绘图逻辑。
- `.nojekyll`：GitHub Pages 配置文件，用于按普通静态网页发布。
- `CNAME`：GitHub Pages 自定义域名配置，当前域名为 `turtleneck.top`。

## 使用方法

1. 打开在线网页，或在本地双击打开 `index.html`。
2. 在“数据填入”页输入或修改实验数据。
3. 直接测量量表格支持“添加一列”和“删除末列”，用于调整每个物理量的测量次数。
4. 砝码质量与标尺读数表格支持“添加一行”和“删除末行”，用于调整拟合数据组数。
5. 点击“生成结果”，网页会切换到“数据生成”页并显示计算结果、拟合图和各项表格。
6. 需要修改数据时，点击“返回修改数据”回到输入页。
7. 点击“打印 / 导出 PDF”，在浏览器打印窗口中选择“另存为 PDF”。

## 计算内容

网页会自动完成：

- 平均标尺读数：
  `b = (加砝码读数 + 减砝码读数) / 2`
- 最小二乘拟合：
  `m = a + k b`
- 直接测量不确定度：
  `ΔA = 1.05σ`
  `u = √(ΔA² + ΔB²)`
- 杨氏模量：
  `E = 8gDLk / (πd²l)`
- 不确定度传递：
  `uE/E = √[(uD/D)² + (uL/L)² + (uk/k)² + (2ud/d)² + (ul/l)²]`

## 默认数据

页面默认载入本次实验数据：

- `L = 0.9955 m`
- `l = 0.0711 m`
- `D = 1.1860 m`
- `d = 0.302 mm`
- 拟合结果约为：
  `E = (1.94 ± 0.07) × 10^11 Pa`
